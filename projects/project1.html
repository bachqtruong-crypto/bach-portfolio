Architecture Design

This architecture was designed for high availability and edge performance. Traffic flows through CloudFront → Application Load Balancer → EC2 instances across multiple Availability Zones.

Architecture Components

High Availability

EC2 instances deployed across multiple AZs behind an ALB.

Security

IAM roles with least privilege access and restricted security groups.

Performance Optimization

CloudFront CDN reduces latency and improves global access speed.
